Welcome to the Anonymity Flow (AF) wiki!

Here you will find in each page a list of all features in that feature set.
All these features are calculated on data that can be extracted from real time network traffic without the need for decrypting messages etc. The aim was to have a tool that could be used in a production environment and be used to do research.
